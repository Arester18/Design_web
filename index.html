7_Otras características de cajas

Aquí veremos como usar: "z-index, outline, position, display".

z.index

z-index Esta propiedad pone una caja encima de la otra, a esta propiedad le podemos poner los valores que queramos, el que tenga mayor valornumerico es el que estará por encima de las demás cajas. La caja padre nunca se puede poner encima de una caja hijo, a menos que le pongamos una position relativa negativa al hijo

Position

Position Existen 4 formas de posicionar: "Static, relative, absolute, fixed, sticky". La posición static es la que viene por defecto, cuando está en esta posición se dice que no está posicionado

1_Relative: Cuando ponemos positon relative podemos mover la caja a otro lugar como si se tratara de un alma, mientras que el espacio que debería tener cuando no tenia esta propiedad se conserva. Podemos mover esta caja fantasma con: "top, right, botton y left", siendo las más importantes top y left

2_absolute: Esto es igual que el relative con la diferencia de que la caja ya no tiene un espacio reservado, pero su ubicación por defecto es donde debería estar la caja si no tuviese esta propiedad. Esta propiedad tambien hace que la caja se ajuste al contenido

3_fixed: Es lo mismo que el absolute con la diferencia de que esta se queda fija en la pantalla del usuario

4_sticky: La position fixed hace que la caja se quede fijo, como si fuera una barra menú. La position sticky se quedrá quieto una vez que el usuario pase por la caja y siga bajando
